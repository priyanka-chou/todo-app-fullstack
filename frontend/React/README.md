# Todo App with Authentication

A full-stack Todo Application built using React, Node.js, Express, MySQL, and JWT Authentication.

## Features

- User Signup & Login
- Email OTP Verification
- Forgot Password
- Reset Password
- JWT Authentication
- Todo CRUD Operations
- User-specific Todos

## Tech Stack

- React.js
- Node.js
- Express.js
- MySQL
- JWT
- Bootstrap
